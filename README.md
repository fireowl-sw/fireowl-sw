### 🚀 Featured Projects (精选项目)

- **✋ [MiniLab-Hand](https://github.com/fireowl-sw/MiniLab-Hand)** - 基于 MuJoCo 仿真器的仿人五指灵巧手强化学习（RL）操作与控制系统，支持手内操纵与动作轨迹可视化评估，提供 [Bilibili 演示视频](https://www.bilibili.com/video/BV1UDj16XEHM/)。
- **🐕 [rl_sar](https://github.com/fireowl-sw/rl_sar)** - 基于 Isaac Lab 的智元 D1 机器人步态控制策略训练与 Sim2Sim 跨引擎（MuJoCo/Gazebo）零样本迁移验证系统，提供 [Bilibili 演示视频](https://www.bilibili.com/video/BV1Kn7g6HEx9/)。
- **🤗 [sugar-code](https://huggingface.co/fireowl-sw/sugar-code)** - 基于 3DGS 的表面对齐场景重建项目，支持利用泊松重建导出高精度 Mesh 网格并导入 Isaac Sim/Isaac Lab 进行物理交互仿真，提供 [Bilibili 演示视频](https://www.bilibili.com/video/BV1bELR6CEET/)。
- **🧠 [minimind-v-model](https://huggingface.co/fireowl-sw/minimind-v-model/tree/main)** - 极简 VLM（视觉语言模型）的复现项目，支持端到端训练、微调与推理使用演示。
- **🚗 [fireowl_sim_drive](https://github.com/fireowl-sw/fireowl_sim_drive)** - 基于 CARLA 仿真器与 ROS2 框架实现的自动驾驶实时点云建图与感知系统。
